# NedoSpotlightSearch

> macOS Spotlight Search analog for Linux.

Quickly Search for files and apps in your system.

Integrations:

- Calculator
- Calendar
- Weather
- Clock
- Contacts
- Notifications
- Browser
- Files
- kind: {kind:jpg, kind:images, kind:video}
- Settings

> No NLP used (yet).
